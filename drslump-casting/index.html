  • ➕ 뤄지 → 츤 츠루린 (초능력, 지적 조력자)
  • ➕ 청신 → 츤 츠루텐 (외계 발명가)
  • ➕ 채미 → 미도리 (차분한 교사/아내)
  • ➕ 반한 → 아카네 (장난꾸러기)
